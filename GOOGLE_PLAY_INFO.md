# Informations pour Google Play Console

## Informations sur l'application

### Nom de l'application
**Index Canada**

### Description courte (80 caractères max)
Professional directory for French & English speakers in Canada

### Description complète
Professional directory for French and English speakers in Canada. Easily find qualified professionals in your area.

Annuaire professionnel pour francophones et anglophones au Canada. Trouvez facilement des professionnels qualifiés dans votre région.

Cette application vous permet de :
- Rechercher des professionnels par catégorie et localisation
- Consulter les profils détaillés avec photos et informations
- Contacter directement les professionnels (appel, email, site web)
- Obtenir des directions vers leurs bureaux
- Découvrir des offres exclusives et des partenaires
- Gérer votre propre profil professionnel

Idéal pour les nouveaux arrivants au Canada qui cherchent des services dans leur langue.

## Détails techniques

### ID de l'application (Package Name)
`ca.indexcanada.app`

✅ **TERMINÉ**: ID unique canadien configuré pour la publication.

### Version
- Nom de version : 1.0.1
- Code de version : 2

### Catégorie suggérée
**Professionnel** ou **Style de vie**

### Audience cible
- Public : Tous les publics
- Tranche d'âge : 18+

## Permissions requises
L'application demande les permissions suivantes :
- **Localisation** : Pour trouver des professionnels près de vous
- **Appareil photo** : Pour prendre des photos de profil
- **Stockage/Photos** : Pour sélectionner des images de profil
- **Internet** : Pour accéder aux données des professionnels
- **Appels téléphoniques** : Pour contacter directement les professionnels

## Assets requis pour le store

### Icônes
- Icône principale : `assets/images/store.png` (512x512 px requis)

### Screenshots
Vous devrez fournir des captures d'écran de l'application en cours d'exécution :
- Au moins 2 screenshots
- Format : Téléphone (portrait recommandé)
- Résolution recommandée : 1080x1920 ou 1440x2560

### Images marketing
- **Feature Graphic** : 1024x500 px (image promotionnelle principale)
- **Icône haute résolution** : 512x512 px (sans transparence)

## Politique de confidentialité
**IMPORTANT** : Vous devrez créer et héberger une politique de confidentialité avant la publication. L'application collecte :
- Données de localisation
- Photos de profil
- Informations de contact
- Données d'usage via Firebase Analytics
